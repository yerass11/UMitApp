This guide helps instructors understand the structure, purpose, and technical value of the UMitApp project.

## Project Purpose
UMitApp demonstrates the integration of mobile development, backend engineering, and cloud services within a healthcare context. It showcases modular iOS architecture, secure payment flows, and real-time communication.

## Learning Objectives
Students learn:
- SwiftUI + UIKit hybrid development  
- MVVM + modular architecture  
- Integration with Firebase and Django REST API  
- Secure payment flow using Stripe  
- Real-time messaging  
- Data modeling and API design  

## Assessment Criteria
- Code quality and architectural consistency  
- API integration and error handling  
- UI/UX quality  
- Documentation completeness  
- Compliance with privacy and security principles  

## Suggested Evaluation
- Review commit history  
- Examine organization of modules  
- Test main user flows (auth, appointments, pharmacy, chat)  
- Review API calls and ViewModel logic
